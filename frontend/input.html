Import Excel ETL

Gunakan fitur ini untuk mengunggah file Excel/CSV data alumni. Sistem akan melakukan mapping kolom, cleaning data, normalisasi nilai kosong, lalu memuat data ke tabel dimensi dan fact.

1. Extract Membaca file .xlsx, .xls, atau .csv.
2. Transform Membersihkan dan menyesuaikan data alumni.
3. Load Menyimpan data ke Data Warehouse.

Kolom tambahan akan diabaikan. Kolom yang sesuai akan dipetakan otomatis ke struktur DW.
